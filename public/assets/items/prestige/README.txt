Icones de la page Prestige.

Pour remplacer une icone stylisee par la VRAIE image du jeu : depose un fichier PNG
ici avec EXACTEMENT le nom indique ci-dessous. Des qu'il est present, il remplace
automatiquement l'icone stylisee (aucun code a toucher, juste deposer le fichier).

  nucleus-neutre.png                  -> Nucleus neutre
  nucleus-parfait.png                 -> Nucleus parfait
  badge-jardin-prestigieux.png        -> Badge Jardin Prestigieux
  badge-de-la-tour.png                -> Badge de la Tour
  badge-world-boss-prestigieux.png    -> Badge World Boss prestigieux
  badge-donjon.png                    -> Badge Donjon
  badge-de-donjon-mineur.png          -> Badge de donjon mineur
  embleme-de-protection.png           -> Embleme de protection

(Le Perin utilise deja sa vraie icone : assets/items/cash/perinparticle.png)

Format conseille : PNG carre, ~64x64 px, fond transparent.
